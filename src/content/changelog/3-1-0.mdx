### Новое

* Добавлено поле для динамического поиска по таблице данных;
* Данные во многих колонках теперь можно сортировать по убыванию или возрастанию значений;
* Добавлена выгрузка в формате XLSX (Excel);
* В таблицу данных парсинга **Wildberries** добавлена колонка с URL видеообложки товара;
* В таблицу данных парсинга **Ozon** добавлены колонки с лучшей ценой товара и количеством найденных лучших цен;
* Добавлена поддержка парсинга страниц продавцов для **Wildberries**.

### Изменения

* Значительно улучшен механизм сбора данных со страниц каталогов **Ozon**;
* Все URL в таблице данных с результатами парсинга стали ссылками! Теперь значения в колонках с артикулом товара, картинками и видео стали кликабельными;
* Вы чаще копируете данные, чем выгружаете в файлы, поэтому мы сделали кнопку **Копировать** доступнее;

### Исправления

* Исправлен сбор цен на **Wildberries**. В рамках A/B теста **Wildberries** показывает некоторым пользователям цену с учётом стоимости доставки;
* Исправлена ошибка парсинга URL титульной картинки товара на **Ozon**.
