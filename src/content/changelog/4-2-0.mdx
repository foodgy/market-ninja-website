### Новое

* Все [акционные страницы](https://www.wildberries.ru/promotions) на Wildberries теперь тоже можно парсить!
    * Включая страницы раздела [Рубли за отзывы](https://www.wildberries.ru/promotions/rubli-za-otzyvy);
* Все страницы из раздела [WB Бренды](https://www.wildberries.ru/wbrands) теперь можно парсить!
* Для **Wildberries** и **Ozon** в колонку **Оплата за отзыв (Review Reward)** выводится количество бонусных баллов/рублей за отзыв;
* В настройки пользователя добавлен пункт **Парсинг => Удалять дубликаты**, который позволяет удалять дубликаты из результатов парсинга;
    * _Актуально для случаев, когда товарная выдача маркетплейса может обновиться во время парсинга и товары с первой страницы могут попатсь на десятую и спарситься снова_;  
* Теперь форму поиска можно активировать с помощью горячих клавиш `Ctrl+K` или `⌘K`;
* Для списков товаров **Wildberries** теперь собирается новый параметр **Реклама (Ad)**, который показывает, продвигается ли данный товар внутренними рекламными инструментами маркетплейса или нет;

### Изменения

* Для страниц брендов и селлеров на **Wildberries** увеличен лимит парсинга (на 101 странице **Wildberries**, как правило уже не тображает товары).
