К спискам товаров относятся страницы результатов поиска, категорий товаров, брендов и списки товаров на странице продавца (селлера).

Расширение поддерживает два режима парсинга списков товаров: **Основные данные** и **Расширенные данные**. Режимы отличаются по объему собираемых данных и скорости работы.

### Режим парсинга: Основные данные

**Что собирает:** Только основные данные о товарах, которые, отображаются на страницах списков товаров.

#### Ozon

| Параметр (Ru) | Описание |
| :--- | :--- |
| Артикул | Артикул товара (уникальный идентификатор на маркетплейсе). |
| Название | Полное наименование товара. |
| Спец. цена | Конечная цена для покупателя с учетом дополнительных скидок (например, за оплату картой Ozon). |
| Цена | Основная цена товара. |
| Старая цена | Цена до применения скидок (зачеркнутая). |
| Скидка | Размер скидки в процентах. |
| Рейтинг | Средний рейтинг товара по отзывам (1-5). |
| Отзывы | Количество оценок/отзывов о товаре. |
| Оплата за отзыв | Количество бонусных рублей за отзыв о товаре. |
| URL | Полный URL карточки товара. |
| Картинка | URL основного изображения товара. |
