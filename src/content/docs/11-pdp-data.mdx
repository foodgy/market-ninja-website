Данные с карточки товара можно собирать в двух режимах: **Только этот артикул** и **Все цвета и варианты**.

### Режим парсинга: Только этот артикул

**Что собирает:** Характеристики товара и другие данные только для одного артикула.

#### Ozon

| Параметр (Ru) | Описание |
| :--- | :--- |
| Артикул | Артикул товара (уникальный идентификатор на маркетплейсе). |
| Название | Полное наименование товара. |
| Категория | Категория, к которой атрибуцирован товар. |
| Спец. цена | Конечная цена для покупателя с учетом дополнительных скидок (например, за оплату картой Ozon). |
| Цена | Основная цена товара. |
| Старая цена | Цена до применения скидок (зачеркнутая). |
| Скидка | Размер скидки в процентах. |
| Лучшая цена | Лучшая цена на Ozon для данного товара. |
| Найдено дешевле | Кол-во предложений на Ozon с меньшей ценой. |
| Размер | Размерный ряд товара. |
| Продавец | Наименование продавца. |
| Рейтинг продавца | Рейтинг магазина продавца (селлера). |
| ID продавца | Уникальный идентификатор магазина продавца (селлера). |
| Рейтинг | Средний рейтинг товара по отзывам (1-5). |
| Отзывы | Количество оценок/отзывов о товаре. |
| Оплата за отзыв | Количество бонусных рублей за отзыв о товаре. |
| Вопросы | Количество вопросов пользователей о товаре. |
| Название размера | Наименование размеров товара. |
| Кол-во картинок | Количество картинок в галерее товара. |
| Кол-во видео | Количество видео в галерее товара. |
| URL | Полный URL карточки товара. |
| Картинка | URL основного изображения товара. |
| Галерея | Список URL всех картинок из галереи картчоки товара. |
| Видео | Список URL всех видео из галереи картчоки товара. |
| Описание | Текстовое описани товара (не Rich-контент). |
| Rich-контент | Наличие Rich-контента на карточке товара. |
| Rich-галерея | Список URL всех картинок из Rich-контента картчоки товара. |
| Rich-описание | Текстовое описание из Rich-контента карточки товара. |

### Режим парсинга: Все цвета и варианты

**Что собирает:** Все данные из режима [Только этот артикул](#карточка-товара-режим-парсинга-только-этот-артикул), а также данные обо всех смежных с данной карточкой товарах (например, другие цвета, веса упаковки и т. п.).
