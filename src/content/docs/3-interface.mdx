Когда вы откроете расширение, вы увидите окно с несколькими основными элементами:

- **Заголовок:** Содержит логотип и название расширения, а также кнопки **Настройки** и **Помощь**.
- **Информация о странице:** Показывает название страницы, URL, иконку маркетплейса, а также количество доступных для парсинга страниц и товаров. Кнопка с лупой позволяет быстро вернуться к вкладке, на которой было открыто окно расширения.
- **Панель управления:**
    - Выпадающее меню для выбора режима парсинга:
        - **Основные данные** и **Расширенные данные** каталогов товаров, результатов поиска, а также списков товаров на странице продавца и бренда;
        - **Только этот артикул** и **Все цвета и варианты** для страницы с карточкой конкретного товара;
    - Кнопка **Собрать данные** для запуска сбора данных (или **Остановить**, если процесс идет);
    - Кнопка **Удалить** для удаления собранных данных;
    - Кнопка **Копировать** для копирования данных в буфер обмена;
    - Кнопка **Экспорт** с выпадающим меню для экспорта данных в формате Excel,CSV или JSON.
- **Полоса прогресса:** Появляется во время процесса парсинга, показывает сколько товаров обработано.
- **Таблица данных:** Отображает все собранные данные в табличном виде.
