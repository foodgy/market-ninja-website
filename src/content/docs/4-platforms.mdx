Расширение работает только на сайтах:

- Ozon ([ozon.ru](https://www.ozon.ru))
- Wildberries ([wildberries.ru](https://www.wildberries.ru))
- Lamoda ([lamoda.ru](https://www.lamoda.ru))

Вы можете собирать данные с четырех типов страниц:

1. **Страницы товара:** Карточка конкретного товара (например, шампуня или кроссовок).
    - Карточка товара [Шапка докер ALEKON Жак-Ив Кусто](https://www.ozon.ru/product/shapka-doker-alekon-zhak-iv-kusto-478494878/) на **Ozon**;
    - Карточка товара [Набор шампунь и бальзам для волос женский бессульфатный](https://www.wildberries.ru/catalog/246438624/detail.aspx) на **Wildberries**;
    - Карточка товара [Кеды Converse Chuck Taylor All Star](https://www.lamoda.ru/p/rtladj766102/shoes-converse-kedy/) на **Lamoda**.

2. **Категория:** Страница со списком товаров в определенной категории (например, **Женская обувь**).
    - Страница категории [Женские купальники](https://www.ozon.ru/category/kupalniki-i-plyazhnaya-odezhda-zhenskie-7540/) на **Ozon**;
    - Страница категории [Женские купальники](https://www.wildberries.ru/catalog/zhenshchinam/kupalniki) на **Wildberries**;
    - Страница категории [Женские купальники](https://www.lamoda.ru/c/3002/clothes-kupalniki/) на **Lamoda**.

3. **Поиск:** Результаты поиска по ключевому слову (например, **кроссовки Nike**).
    - Результаты поиска по запросу [кроссовки Nike](https://www.ozon.ru/category/obuv-17777/?category_was_predicted=true&deny_category_prediction=true&from_global=true&text=%D0%BA%D1%80%D0%BE%D1%81%D1%81%D0%BE%D0%B2%D0%BA%D0%B8+Nike) на **Ozon**;
    - Результаты поиска по запросу [кроссовки Nike](https://www.wildberries.ru/catalog/0/search.aspx?search=%D0%BA%D1%80%D0%BE%D1%81%D1%81%D0%BE%D0%B2%D0%BA%D0%B8%20Nike) на **Wildberries**;
    - Результаты поиска по запросу [кроссовки Nike](https://www.lamoda.ru/catalogsearch/result/?q=%D0%BA%D1%80%D0%BE%D1%81%D1%81%D0%BE%D0%B2%D0%BA%D0%B8%20nike&submit=y&gender_section=women) на **Lamoda**.

4. **Страница бренда:** Список товаров определенного бренда (например, все товары **Adidas**).
    - Страница бренда [Adidas](https://www.ozon.ru/brand/adidas-144082850/) на **Ozon**;
    - Страница бренда [Adidas](https://www.wildberries.ru/brands/adidas) на **Wildberries**;
    - Страница бренда [Adidas](https://www.lamoda.ru/b/1061/brand-adidas/) на **Lamoda**.
