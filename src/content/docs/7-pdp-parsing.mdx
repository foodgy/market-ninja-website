Этот сценарий используется, когда вам нужно проанализировать конкретный товар и его варианты.

### Как запустить сбор

1. Откройте страницу конкретного товара;
2. Запустите расширение;
3. Выберите режим парсинга;
4. Нажмите кнопку **Собрать данные**.

### Режимы парсинга

#### 1. Только этот артикул

Моментально собирает данные только о текущем выбранном цвете/размере товара, который вы видите на экране.

<Video src="/videos/demo-pdp-current-sku-only.mp4" autoPlay={false} />

#### 2. Все цвета и варианты

Собирает данные о текущем товаре, а также обо всех его вариациях (другие цвета, модели в этой же карточке).

<Video src="/videos/demo-pdp-all-colors-and-variants.mp4" autoPlay={false} />

*   **Как это работает:** Расширение находит ссылки на другие цвета в карточке и по очереди собирает данные о каждом из них.
*   **Результат:** В таблице появится несколько строк — по одной на каждый вариант товара. Это удобно для анализа ассортимента одной модели.
